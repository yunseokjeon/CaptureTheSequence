금융 시장 데이터를 활용해 거래전략에 대한 최적 베팅 비율을 계산해 내는 웹앱 입니다.  데이터로 부터 기회를 포착해 낸다는 의미에서 'Capture The Sequence' 라고 이름을 지었습니다.

- Frontend : React
- Backend : Spring Boot / JPA
- Database : MySQL
